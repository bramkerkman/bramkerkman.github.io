De CPU

In het moederbord van je computer ligt de CPU wat staat voor central processing unit of wel processor is een belangrijk stuk hardware in de computer. Het wordt vaak beschreven als het hart van een computer. De CPU is vooral bezig met twee dingen. Het maken van berekeningen en het besturen van taken. De processor doet dit door transistors. Transistors werken als schakelaars die een logische functie kan opbouwen. De snelheid van de processor heeft ook invloed op de snelheid van de taken die worden uitgevoerd. Dit wordt aangegeven in verwerkingssnelheid. Ook is er een kloksnelheid. Hiermee worden het aantal pulsen berekend die de processor uitvoert.
